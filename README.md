# 🚀 RENT EASE – Smart Rental Marketplace

**RENTEASE** is a modern web-based rental marketplace that connects property owners and customers on a single platform.
The system allows owners to list rental properties such as rooms, houses, and lodges, while customers can browse listings, view property details, and communicate directly with owners.

The platform focuses on **transparency, location visibility, and real-time interaction**, making the rental process simpler and more efficient.

---

# 🌟 Key Features

### 🔐 Secure Authentication

* User registration and login system
* JWT-based authentication
* Role-based access (**Owner / Customer**)

### 🏠 Property Listing System

* Owners can create and manage property listings
* Supports multiple property types:

  * Rooms
  * Houses
  * Lodges
  * Apartments
  * PG accommodations
* Add property details such as price, amenities, square footage, images, and description

### 📍 Google Maps Integration

* Property location displayed using **Google Maps**
* Customers can view the exact property location
* One-click navigation to open directions in Google Maps

### 💬 Real-Time Chat

* Direct chat between property owners and customers
* Helps clarify property details quickly
* Improves communication and decision making

### ⭐ Rating & Review System

* Customers can rate properties
* Improves transparency and trust between users

### 🤖 AI-Powered Description Generator

RENTEASE integrates **Google Gemini AI** to help property owners generate professional property descriptions automatically.

When an owner enters property details such as amenities, size, and location, the system uses the **Gemini API** to generate a well-structured description for the listing.

**Benefits:**

* Saves time for property owners
* Generates professional property descriptions
* Improves listing quality
* Makes property listings more attractive

### 📱 Responsive User Interface

* Clean and modern design
* Fully responsive layout
* Optimized for mobile and desktop devices

---

# 🧠 Problem Statement

Finding suitable rental rooms, houses, and lodges is often difficult due to scattered platforms, outdated listings, and lack of communication between property owners and tenants. Tenants often struggle to verify property details, availability, and location before making decisions.

Property owners also face challenges in managing listings and responding to customer inquiries efficiently. Therefore, a centralized platform is required to simplify the rental process and provide reliable communication between owners and customers.

---

# 💡 Proposed Solution

RENTEASE provides a centralized rental platform where property owners can create detailed property listings including amenities, pricing, square footage, and location using Google Maps integration. Customers can browse available properties, view accurate information, and communicate directly with owners through real-time chat.

Additionally, the system integrates **AI-powered description generation using Gemini API**, which automatically creates professional property descriptions based on listing details.

This improves listing quality, reduces effort for owners, and enhances the rental experience.
